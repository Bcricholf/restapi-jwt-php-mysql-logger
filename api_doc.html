Règles générales

Les codes d'erreurs se basent sur ceux de l'API

Ecrire un message de log

Method: POST

Auth required: NO

Data constraints

{
    "name":"writeLog",
    "param":{
        "message" : "message de log",//Obligatoire
        "level" : "error"//Obligatoire
    }
}

Success Response

Code: 200 OK

Content example

    {"status":200,"result":{"result":"Log add."}}

Générer une call trace

Method: POST

Auth required: NO

Data constraints

{
    "name":"generateCallTrace",
    "param":{
        "level" : "info",//Obligatoire
        "giveAllHistory" : "true"//Obligatoire
    }
}

Success Response

Code: 200 OK

Content example

    {"status":200,"result":{"result":"Call trace generate and add."}}
